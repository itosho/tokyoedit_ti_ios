Tokyo Edit（β）

Concept

『Tokyo Edit』は"日常に潜むパラレルワールド"というコンセプトをもとにつくられた地下の世界と地上の世界とを繋げるモバイルコンテンツです。

各ムービーでは、潜望鏡を覗きこむような感覚で地下鉄が走っている地上の風景をみることができます。アプリを使って、暗い地下鉄の車窓からぼんやりと映し出されるもう一つの世界を体験してください。

How To Use

All Tokyo Edit
すべての動画をストリーミング再生で視聴できます。また、各動画は再生画面のメニューからダウンロード可能です。
お気に入りの動画をみつけて地下鉄の車窓をお楽しみください。

My Tokyo Edit
ダウンロード済の動画をオフラインで視聴できます。また、再生画面のメニューから走行ルートも閲覧できます。
尚、動画を削除したい場合は各端末のギャラリーから削除可能です。
